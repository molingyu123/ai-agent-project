# AI Agent Production Project

## Overview
生产级AI智能体系统，支持：
- 对接原始老业务系统
- 定时监听更新数据
- 知识问答（RAG）
- 项目数据分析（Pandas + Excel报告）

## Quick Start
```bash
pip install -r requirements.txt
python src/main.py
```

## Features Implemented
- Legacy integration with config-driven API
- Scheduled monitoring
- Knowledge QA module
- Data analysis with report generation
